## Practice

### Media Object

Study the following media object.

![Media Object](media.png)

HTML:
```html
<div class="container">
	<div class="graphic"></div>
	<div class="content">Content here</div>
</div>
```

CSS:
```css
.container {
	background-color: red;
	width: 400px;
	display: flex;
}

.container .graphic {
	width: 50px;
	height: 50px;
	background-color: blue;
}

.container .content {
	flex: 1;
	background-color: yellow;
}
```
