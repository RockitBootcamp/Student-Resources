### #1: Create ul that has each key/value pair as an li
The PHP code:
```php
$items = '';
foreach($pet as $key=>$value) {
    $items .= "<li>$key: $value</li>";
}
```
The HTML code:
```html
<!-- This -->
<ul><?= $items ?></ul>

<!-- Or This -->
<ul><?php echo $items ?></ul>
```
